# Note taking app with backend 

# Description

A full-stack note-taking app built with Node.js and Express. Users can create, edit, complete, and delete notes through a simple dashboard UI, with notes stored persistently in a JSON file on the server.



# Installation

1. Navigate into the project folder:
   cd notes-app
2. Install dependencies:
   npm install
3. Start the server:
   node server.js
4. Open http://localhost:3001 in your browser


# Usage

- Type a task into the "Enter a task" field and click "Save Note" to create a new note.
- Tick the checkbox next to a note to mark it as complete (shown with strikethrough text).
- Click "Edit Note" to change a note's text.
- Click "Delete" to remove a note permanently.
- All notes are saved to data.json on the server, so they persist between page refreshes and server restarts.
